Backend

Choose your local LLM inference backend. Each backend has its own default model and URL — switch freely, your settings persist per provider.

To use Ollama: Install from ollama.com, then run in a terminal: open /Applications/Ollama.app or ollama serve

Used as the initial selection when no prior model was chosen. Pull with ollama pull <model>.

To use MLX: Install MLX LM, then start the server with one of these copyable commands.

When enabled, the main chat screen shows a button under the model picker that starts MLX in the background and writes output to timestamped log folders.

Install MLX LM
pip install mlx-lm
Start MLX server
python -m mlx_lm.server --model mlx-community/Llama-3.2-3B-Instruct-4bit

Replace mlx-community/Llama-3.2-3B-Instruct-4bit with your chosen model id.

Must match the --model id you passed to mlx_lm.server.
